Registering an exemption on the PRS Exemptions register

The following is a list of valid exemptions:

  • If there are no recommended improvements for the property
  • If the landlord will incur a cost by installing the recommended improvements
  • If the recommended wall insulation system cannot, or should not, be installed
  • If consent cannot be obtained from the tenants and / or relevant third parties
  • If installing the improvements would results in a devaluation of the property
  • If you recently became a landlord

If any of these exemptions apply, you need to file a report with the exemption register. You will need to include relevant supporting evidence.
